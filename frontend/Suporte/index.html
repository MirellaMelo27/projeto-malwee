Manual do Sistema

Bem-vindo ao Manual do Sistema!

Aqui você encontrará todas as instruções para utilizar o painel, configurar relatórios e interpretar gráficos.

1. Acesso ao Sistema

Para acessar o sistema, entre com seu usuário e senha na tela de login. Caso tenha problemas de acesso, contate o suporte técnico.

2. Navegação

O menu lateral contém as seções principais: Visão Geral, Detalhes, Análise e Configurações.

Você pode alternar entre os temas claro e escuro nas configurações do sistema.

3. Gráficos
  • O sistema possui vários tipos de gráficos.

Mais informações serão adicionadas conforme o sistema for atualizado.
